# Change Log

All notable changes to the "copymate-pro" extension will be documented in this file.

## [0.1.0] - 2026-05-15

### Added

- Browse workspace files from the VS Code sidebar
- Filter files and folders using `.copyignore` support
- Copy structured file contents to clipboard in LLM-friendly format
- Copy folder tree as ASCII diagram
- Select All / Unselect All files via toolbar
- Add files or folders to `.copyignore` via right-click or toolbar button
- Auto-reload tree when `.copyignore` is saved
- Progress notification while reading large file sets
- Built for sharing code context with LLMs like Claude, ChatGPT, and GitHub Copilot
